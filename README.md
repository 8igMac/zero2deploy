# zero2deploy
Back to basic: machine learning from zero to deploy.

之後一系列的訓練課程我們將要來解決一個 NLP 的經典問題，從爬取資料、儲存資料、資料前處理、
套用模型、一直到專案後期的 demo UI、docker 佈署。

這個專案會請你們用 git 來做版本控制，並把專案上傳到自己的 GitHub。
